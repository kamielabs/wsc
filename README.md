# WSC — Workspace CLI (v0.1)

WSC is the core development CLI used across KamieLabs projects.  
It provides a unified environment for managing workspaces, configuration, runtime, logging, tooling and project scaffolding.

> ⚠️ This project is in early active development (v0.1).  
Expect rapid changes and heavy refactoring.

## 🔗 Documentation
See the shared KamieLabs documentation here:  
https://github.com/KamieLabs/.docs

## 🛠 Local development

pnpm install
pnpm dev
